### redmine的安装配置

参考redmine官网的install Guide 部分：https://www.redmine.org/projects/redmine/wiki/RedmineInstall

主要需要的几个部分：

- ruby and rail (有版本要求)
- MySQL

步骤：

1. 用MySQL设置一个数据库，配置好数据库的链接
2. 用ruby的包管理工具gem安装一些必要的库
3. 然后是一些无关紧要的配置
4. 测试

PS：该网站还有一些对于windows用户的提示notes
